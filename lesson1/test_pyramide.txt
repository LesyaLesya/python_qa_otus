Цель обучения - научиться писать эффективные и красивые авто-тесты, перейти из ручного тестирования в автоматизированное и внедрить автоматизацию тестирования в текущий проект.
По пирамиде автоматизации на текущей работе:
1. Некоторые разработчики иногда пишут юнит тесты
2. API тесты не автоматизированы. Проверяются через Swagger
3. Есть один автоматизатор, который пишет только UI тесты
